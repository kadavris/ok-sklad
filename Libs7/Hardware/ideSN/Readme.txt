This is simple console application that extract the serial number from
the first IDE HDD.

Require Windows 95 OSR2, Windows 98, Windwos 98 SE, Windows ME,
Windows NT4 or Windows 2000.

Windows 9x:
  SMARTVSD.VXD must be installed: just copy it from \windows\system to
  \windows\system\iosubsys and reboot.

Run Build.bat to compile the sample application.
Start BIN\IdeSN.exe from command prompt.
