Quick filter bar

At the top of the window there is a bar with a few sorting options available to you. You may select a range of dates to display, choose specific business partner or display "All" and narrow down your selection by document status: All, posted or non-posted

Documents list columns

The topmost document table columns are:

• - icon of Goods dispatch slip document type.
• - indicates that the document has been posted
•No. - Serial number of the document
•Date - date of last save
•Business partner - customer, recipient of goods
•Amount, USD - Total goods selling price in system base currency
•Amount in currency - Total goods selling price in document's currency
•Paid - indication that these goods have been fully or partially paid by recipient and there exists a linked incoming payment(s)
•Seller company - your company

Status bar at the bottom of the screen shows a document counter and a total amount in the base currency

Documents list popup menu

Right-click of a mouse will bring up a drop-down menu.

•Add - add a new goods dispatch slip
•Add using current - creates a new goods dispatch slip based on the selected one. All information will be transferred from the current document so you won't have to enter repetitive data.
•Delete - this option is only available when the document is not posted and no other documents linked to it (such as payments, etc) exist.
•Refresh - refreshes the list of documents. This is helpful when the system is in multi user mode
•Print/Preview - displays the document as it will appear when printed
•Add payment against document - creates a payment document linked to the selected document.
•Create invoice - creates invoice based on the properties of the selected slip
•Unpost - makes a selected document not posted, ultimately removing them from accounting balances
•Show documents with zero balance - toggles display to show documents with zero amounts. Such documents can be used for example for item reservation at warehouse pending price calculations
•Apply filter - Toggles the date, business partner and state filters activity
•Set up columns - choose from the list which columns will be displayed
•Export selected to Excel - exports selected data from the document to Excel file
•Edit - used to edit the document. For consistency reasons this option is only available when the document is not posted

 

 

Selected document popup menu

 

When you highlight the document in the main window, the bottom information panel changes according to the data of this slip.

Under "Items" tab the list of items to be shipped with detailed information on each item is displayed.

 

The columns of the table are as follows:

•# - sequence number
•Reserved - indicates that this item has been reserved at warehouse, so it cannot be sold to anyone else. This indication is for "not posted" documents only. Posting the document will remove items from warehouse as sold.
• icon is for goods and icon is for services
•Product/Service - item name
•Quantity - goods quantity
•Unit - item unit of measure
•Price, USD - price per unit in document's currency
•Price in currency - price per piece in base currency
•Amount net - total for all quantity of items in document's currency
•Amount in currency - total for all quantity of items in base currency
•Discount - percent discount given
•Serial numbers - displays serial numbers if products under this item have unique identification
•Warranty period - if warranty for this item is provided, displays warranty period. will be shown here

 

Right-click menu:

•Product information - allows you to review details of the selected product item
•Product movement details - presents income, sales and movement of the product between warehouses.
•Product reserve details - displays reservation data for the product
•Display list of substitute products - displays the list of available substitute products
•Print/Preview - allows you to print a warranty card. You will be able to select one or more items to include in the warranty list.
•Create invoice - brings up a window for invoice editing. form with automatically added all selected items from current document
•Find in products directory - displays product directory under "Directories" category
•Copy - copies selected item details as plain text with header into clipboard to be used in different applications.
•Set up columns - choose from the list which columns will be displayed
