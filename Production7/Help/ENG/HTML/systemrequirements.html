Minimal requirements:

Operating system: Microsoft Windows 2000 SP4, XP SP2 or Vista SP1.

CPU: Single 500 Mhz for Windows 2000 and XP*

RAM: 256 Mb for Windows 2000 and XP*

Video: any video adapter coupled with display that both support 1024x768 resolution

LAN: any working Ethernet/Wi-Fi/WiMax adapter

Hard drive free space: 50 Mb

Internet connection: 128 Kbit or faster for Web Stores Management option (for small or no product images and quantity)

 

Optimal performance system:

Operating system: Microsoft Windows 2000 SP4, XP SP2 or Vista SP1.

CPU: Dual CPU or Dual Core 2 GHz for Windows 2000 and XP*

RAM: 1 Gb for Windows 2000 and XP*

Video: any video adapter coupled with wide screen 17 inch display that both support 1280x1024 resolution with at least Hi-color (16 bit, 65k colors)

LAN: 100Mbit Ethernet or WiMax adapter

Hard drive free space: 100 Mb

Internet connection: 1 Mbit or faster for Web Stores Management option

 

* Note: Windows Vista requires approximately 2 to 4 times more RAM and CPU resources, especially when Aero interface extension is enabled

FYI: It is not available hence not enabled by default only in Vista "Basic" edition.

OK-Inventory can run on all Windows Vista editions.

 

Although some tests prove that our software is able to run on earlier Windows versions such as Windows 98 and Windows Millennium,

we do not guarantee the flawless operation on these systems nor do we provide any support in case of problems.
