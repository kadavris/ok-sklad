Event Log

 Top  Previous  Next

 Clicking on «Event Log» will bring up an informational screen containing the data on all operations that took place in the program sorted in a chronological order.

By default, all events in the event log will be displayed from the initial moment to the current date on all categories of the program indicating all users who performed these operations. In other words, even log represents the history of all company activity.

 

You may change the parameters of the event log to specify time periods or select a specific category (e.g. incoming payments or tax documents etc), sort the data per specific user after you have specified the required selection criteria in the appropriate fields.
