Unposting rules

 Top  Previous  Next

If you want to unpost the document, you should keep in mind the following:

Unposting is only possible if the product being unposted remains at warehouse and not being shipped or transferred. This means that nothing was registered in another document as sold, reserved, moved to another warehouse or written-off. This restriction exists for the purpose of maintaining financial integrity in the system and prevent any extras or shortages at.
A warning message will be displayed when unposting document..
In order to proceed anyway, the items listed should not appear in any other kind of posted documents. The list of linked documents can be viewed using "Linked documents" form.

For "Goods transfer slips" and "Write-off" types it is sufficient to simply change their state to unposted.
Before unposting "Order" and "Goods dispatch slip" documents you need to make sure that the items in this documents are not reserved.
To cancel reservation you need to right-click on selected items and select "Make available" in the drop-down menu.
Note that there is no need to reset all items on the list, only the ones that present in the original receipt document.
