  'Settings': - it is main options for the OK-Inventory

  Users - these settings will define the names and access rights for users who will work with the database.

  User groups - settings in this section will allow you to combine the users according to specific criteria - common to this group of users.

  User monitor - allows the system administrator of the program to get the data on all users currently working in the program.  

  Databases - contains the information on existing databases with database paths on computer hard drives.

  Event Log - stores the history of all operations and changes made in the program since the beginning of use up to current date with sorting options by users or user groups.
