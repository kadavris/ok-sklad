This category contains the tools required to create and print all primary documents such as invoices, goods receipt and goods dispatch slips, price lists etc. It also allows the user to store the information on supplier / customer contracts to be used in tax documentation. Each type of the document has an icon associated with it, which relays the purpose of the document. Double-click on an icon will expand the complete list of the documents in the present category.

 

Way-bills / Slips:

• Goods Receipt Slips - this sub-item contains the information on all goods receipt slips of the company sorted by date. Usage of this tool (registration of all goods receipt slips) allows the user to register all goods coming to company's warehouses as well as transactions with suppliers.
• Invoices - this sub-item contains the information on all invoices issued for all goods or services. This tool will allow the user to generate and print out invoices for goods or services to customers. The program also offers a convenient way to issue goods issue using "invoice-waybill" system. Right-click of a mouse or pressing F2 button will bring up a menu where you will be able to select a "Post" option, after that a message will appear that a goods dispatch slip will be created. You will need to either agree or cancel the operation.
• Goods Dispatch Slips - this sub-item contains the information on all goods dispatch slips issued for goods  sorted by date. This tool allows the user to generate and print out goods dispatch slips. It also provides an option to keep track of all goods issued from company warehouses. There is a convenient sequence "goods dispatch slip - tax slip", which you may use to generate a tax slip, right-click of a mouse will bring up a window where you will be able to select this option.
• Tax Slips - this sub-item contains the information on all issued tax slips, sorted by date. As it was shown before, tax slips may be generated and printed out using "goods dispatch slip - tax slip" sequence. Using of this sub-item will allow the user to issue one or several (if necessary) tax slips for each of the documents independently.

 

  Payments:

• Incoming payments - this sub-category contains the information on all payments received by the company (system user) from customers by wire transfers, checks or in cash.
• Outgoing payments - this sub-category contains the information on all payments sent by the company (system user) to suppliers by wire transfers, checks or in cash. Here the user will also be able to generate and print out invoices.

 

  Purchase and Sales Contracts:

• Purchase Contracts - this sub-category contains the information on all relevant purchase contracts between companies (system users) and suppliers. It allows the user to promptly obtain the information on terms and conditions of existing contracts.
• Sales Contracts - this sub-category contains information on all sales contracts concluded on behalf of the company between system users and the buyers. It allows the user to promptly obtain the information on terms and conditions of existing contracts.

 

Trash Bin - This item allows the user to store deleted documents in the database until the user decides to remove them permanently. The user is able to restore documents from Trash Bin into their original locations.
