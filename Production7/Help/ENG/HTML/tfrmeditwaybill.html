Main properties

Main properties of the document being edited are displayed at the top left corner of editing window:

•Post document - posts the document into the system (the document becomes part of company's financial balance)
•Vendor  - the name of the business partner, from whom you receive/purchase goods. By clicking on button you can select the individual or company from the list of your business partners. You can also enter a desired document number into the "No." field instead of the default sequence number and edit the date and time of actual event if, for example, you wish to register a shipment that was made on a previous day.
•Contract - identification number of the contract this slip was based on. Leave blank if no contract exists.
•Warehouse - name of warehouse where the goods are being received to.
•Currency - can be set different from system default currency (e.g. if foreign currency was used for goods purchases). The "Rate" field is used to show the exchange rate between system and selected currency.
•Received by - displays the name of receiver of goods..

This section used to register payments that have already been issued.
You will need to provide the ID number of the document, payment type, employee name responsible for the payment, payment type, amount and currency.

These buttons on top allow you to add and remove goods and services from the list

Clicking on down arrow of "Add" button allows you to select the type of goods or services to be added - single or multiple goods / services.

Item list

• icon is for goods and icon is for services
•No - sequence number
•Model - Model number as appears under "product description"
•Bar code - Bar code on the label. Available with "Bar codes" option
•Quantity - Goods quantity
•Unit - Unit of measure or type of service
•Price VAT incl. - purchase price including VAT added (if applicable at your location)
•Amount VAT incl. - total for all quantity with VAT included (if applicable at your location)
•Serial number - list of serial numbers if applicable to this product type.

Items right click menu

•Add - same as "Add" button atop of item list
•Delete - same as "Delete" button atop of item list
•Select all - selects all items on the list
•Product information - shows modal dialog with selected product properties
•Display non-rounded values - toggles display of raw numbers for calculated amounts. Roundup settings can be changed using system setup screen
•Set up columns... - allows you to decide what kind of information will be  displayed in this table. For example you can turn off display of product/service name if you are using vendors' information instead.
•Edit - allows you to edit item properties - same as double-clicking on item
