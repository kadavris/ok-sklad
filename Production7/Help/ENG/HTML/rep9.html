Goods movement report

 Top  Previous  Next

This report is intended for preview and printing of the data on goods received and issued indicating document ID numbers and dates, business partner names and current stock balances at warehouse

You can start generating this report by clicking on icon or pressing at top toolbar or pressing "Ctrl + Р" or simply double-clicking on report name. A drop down window will allow you to define report selection criteria. You will need to specify the date for which the report is being generated or select it from the drop-down calendar, or enter it manually in the field. If you wish to view the complete history of goods movement, you will need to simply remove the dates from the "from" and "to" fields.

Then you need to specify the name of the warehouse for which the report is being generated. You can select it from a drop-down menu or by clicking on icon. Press "Enter" to confirm your selection or double-click on the line with the warehouse name. If you are interested to get the report on all warehouses, select "All" in the warehouse field.

The next selection criterion is "product name" or "product group", which you can choose using drop-down or by clicking on icon. It will display a list of products where you can select the one you require by pressing "Enter" or double-clicking on the product name
