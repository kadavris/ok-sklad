Document Log

 Top  Previous  Next

This report contains the information on all operations performed by the company -- user of this program indicating document ID number, date, amount for the document and business partner name. Selection criteria are defined by time period (from date to date) and document type.

You can start generating this report by clicking on icon on the top toolbar or by pressing "Ctrl + Р" or double-clicking on the report name. A drop-down window will allow you to define report selection criteria. You need to specify the date for which the report is being generated either by using scroll bars or entering the date in the appropriate fields manually. If you need to see the complete log for all period of operation, leave the date fields blank.

You need to specify document status for which you wish to receive the report. Document status is defined using scroll bars (posted or non-posted documents). If you are interested in all documents (both posted and non-posted), select "All" in the "Status" field.

After you've selected document status, you need to select from the list of all document types that should be included in the report. You will need to simply put a check mark on the list of all available documents selecting the one you require (goods receipt notes, goods dispatch notes, accounts, outgoing payments, incoming payments etc.)

  When all parameters of this report are specified, you may click on "Preview"  button or icon and the document will be display as it will appear printed.

In order to print the report click on or press  "Ctrl + Р".
