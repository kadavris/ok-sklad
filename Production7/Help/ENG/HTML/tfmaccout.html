Main table (document list)

•  icon - invoice type of document icon.
• icon - for Completed (fully paid and shipped goods) invoices. Blue icon is for documents still "in Progress"
•No. - invoice sequence number
•Business partner - Customer's name
•Amount, USD - Total for this document in the system default currency (USD in this example).
•Amount in currency - Total for this document in document's set currency.
•Document balance - The unpaid amount remaining for this document (in system default currency)
•Status - In progress - for partially completed payment/shipments, Completed - payment and shipment have been completed

Main table right-click menu
•Add - creates a new invoice
•Add using current - creates a new invoice based on currently selected invoice. It copies for you the main properties, sets new document name and also copies all product and service items with quantities, prices etc. It is very useful for regular supplies for a permanent customer.
•Delete - removes selected document(s) from database
•Refresh - refreshes the list of documents. Useful in multi-user mode when multiple users are connected to the same server and the database is busy updating..
•Print/Preview - allows you to print or view the invoice as it will appear printed.
•Add payment against invoice - creates an incoming payment document linked to selected invoice
•Ship all items - When invoice is In Progress state you can create Goods dispatch slip containing all information about the client and the goods will be shipped. This action switches the status of invoice to "Completed". You will still be able to remove items from Goods dispatch slip, making partial shipment, in this case the invoice will remain "In Progress".
•Show completed and paid invoices - toggles display of processed document. Used to sort out  invoices if the number of documents on the list is high and you only need to see the documents that require attention.
•Apply filter - toggles use of the search fields on top and bottom of program window
•Set up columns - allows you to set up columns that will be displayed
•Export selected to Excel - creates a Microsoft Excel compatible spreadsheet with the list of selected documents and general properties
•Edit - allows you to edit document properties, same as double-clicking on the document

Bottom pane.

Items list for selected invoice

 

• icon is for goods and icon is for services
•icon indicates the item on reserve
•Product/Service - item name
•Shipped - quantity of the item shipped (included in linked Goods dispatch slip(s))
•Reserved by invoice - quantity reserved on invoice
•To ship - quantity of items pending shipment to customer
•Quantity - total number of items
•Unit - item unit of measure
•Price Net - Price excluding VAT
•Price including VAT - item price + VAT
•Amount Net - Total for all items, excluding the VAT
•Amount including VAT - total for all items including VAT accounted

Items list right-click menu
•Product information - overview of the selected product properties. Same as the Product properties editing form, but in read-only mode.
•Product movement details - displays modal dialog with the list of documents containing selected product as one of the items.
•Product reserve details - the list of reserved quantities for all warehouses
•Display list of substitute products - If set in product properties, you will see the list of products in case the required product is not available.
•Ship items - Same as Ship all items in the main table right-click menu. When invoice is In Progress state you can create Goods dispatch slip containing all information about the customer and the goods to be shipped. Switches invoice status to "Completed". You can still remove items from Goods dispatch slip to make a partial shipment. In this case your invoice will remain "In Progress".
•Remove from reserve - used to cancel reservation for non-shipped items making them available at warehouse for other customers.
•Find in products directory - brings up a dialog window with selected product properties.
•Copy - copies all selected item data into clipboard in plain text format
•Set up columns - allows you to specify the columns that will be displayed

 

General invoice properties
•# - ID number of selected document and date it was last edited
•Invoice valid through - Expiration date. Used for information purposes
•Customer section - Used to specify customer information such as name, address, etc.
•Seller company - the name of your company where sale is originated
•Amount - Invoice total for all items
•Based on - the name of document this invoice is based on (if applicable)
•Status - Posted or unposted

Associated documents list

• icon in this example indicates Goods dispatch slip as stated in the document type column. Displays respective icons for other document types
• icon indicates that this document state is Posted
•No. - ID number of the document
•Amount - total for all items included in this document
