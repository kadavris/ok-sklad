Main properties

•No. - Document ID number. Default is sequence number with user defined prefix and suffix from system options
•Customer - you can either specify customer name or click on button to select customer from the list of business partners already existing in the database.
•Contract - ID number of contract or agreement this invoice based on. You may click on button to select it from the list of contracts already existing in the database.
•Currency - you can select different currency for this document for for foreign sale, etc.
•Invoice valid through - specified expiration date for this invoice, for example in case of a price change after a certain period.

Item processing

These buttons on top allow you to add and remove goods and services from the list

By clicking on down arrow on the right side of Add button you can choose what type of goods to add: single or multiple, products or services

Item list

• - goods icon, - service icon
•Reserved - indicates that item was reserved at the warehouse. You can reserve goods even if the whole document is "unposted", to prevent the goods from being sold via another document.
•No. - sequence number
•Product/service - the name of product or the service for this item
•Hours - required amount of time to perform the service
•Quantity - quantity of products/services to be sold
•Unit - Unit of measure of product / service
•Price, VAT included - net price in your default currency w/o VAT
•Amount VAT included - Total for all quantity of goods in default currency (if VAT is applicable)
•Amount in currency, USD - Total for all quantity of goods in the currency selected for the document. Currency three-letter code will be shown (USD, EUR etc.)

Item right-click menu
•Add - same as "Add" button atop of item list
•Delete - same "Delete" button atop of item list
•Select all - selects all items on the list
•Reserve (all)/Make available (all) - reserves or cancels reservation for selected items.
•Serial numbers - brings up the dialog with the list of serial numbers of individual items available for inclusion into this document
•Set discount - applies discount for selected items
•Product information - shows modal dialog with selected product properties
•Service details - displays modal dialog with detailed information on selected services
•Display non-rounded values - toggles display of raw numbers for calculated amounts. Roundup settings can be reached at the system setup screen
•Set up columns... - allows you to decide what kind of information will be displayed in the table. For example you can turn off display of product/service name if you are using vendors' information instead.
•Edit - allows you to edit item properties - same as double-clicking on item
