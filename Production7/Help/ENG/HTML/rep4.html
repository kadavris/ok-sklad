Goods received report on suppliers

 Top  Previous  Next

This report contains the information on all purchasing documents received from suppliers indicating document issue date, document ID numbers, product range, costs and prices of the products and services received from suppliers. Selection criteria include time period (from date to date) and supplier name.

You need to start generating this report by clicking on icon on the top toolbar. Alternatively you can use "Ctrl+P" combination or simply double-click on report name. A drop-down menu will allow you to determine report selection criteria. You can specify time period required for the report by clicking on date selection button or enter it manually into date field. If you require a report for all period of your company operation, leave the fields blank

Then you need to enter supplier name for which the report is being generated. You may select a customer using either scroll bars or by clicking on icon. Confirm your selection by pressing "Enter" or double-clicking on a selected supplier name and it will appear in the "supplier" field. If you need the report on all customers, select "All" from the list.  

  When all parameters of this report are specified, you may click on "Preview"  button or icon and the document will be display as it will appear printed.

In order to print the report click on or press  "Ctrl + Р".
