*****  Версия 1.4.4 *****
+  Добавлена новая схема работы с товаром по серийным номерам: теперь при приходовании
   товара есть возможность выбора ввода серийников для каждой единицы товара в отдельности 
   (так, как и было) - при этом выполняется разбивка партии на N партий по кол-ву товара, либо 
   же определение серийных номеров для всей партии из N единиц товара одной строкой - в этом 
   случае разбивка партии не выполняется. Такая схема позволяет реализовать алгоритм работы,
   при котором серийные номера товара могут указываться только при отгрузке товара клиенту, т.е.
   нет жесткого контроля по партиям. Например, при приходовании товара серийные номера можно не
   указывать вообще, а при списании вводить их вручную.

   простые и накопительные, для последних введена возможность определения пороговых значений и в 
   случае преодоления порога по сумме выполняется автоматический пересчёт скидки по карте.
   Каждая карта относится к группе, которые пользователь может организовать в любом нужном ему 
   порядке (аналогично группам товаров и услуг). Для каждой из групп есть возможность определения
   автонумерации карт внутри группы, а также перенесение параметров скидки из группы на каждую 
   вновь созданную карту. В справочнике добавлена возможность просмотра истории покупок по карте, 
   а также формирование соответствующего отчёта.
   В бесплатной версии программы есть ограничение по количеству дисконтных карт: пользователь 
   может создать и работать не более чем с 5-ю картами.
+  Расходные накладные, счета.
   * При добавлении позиции в документ добавлена возможность указания для неё скидки либо вручную
      (как это было ранее), либо из дисконтной карты.
   * В окне "Добавить накладную/счёт" добавлены колонки "Номер карты" и "Держатель".
+  Склад. Информация о резерве. В списке больше не будут выводиться записи с нулевым резервом, 
   которые появлялись в результате отгрузок по счетам.
+  Расходные накладные, счета. В журнале документов на закладке "Позиции" добавлена колонка "Резерв",
   идентифицирующая наличие резерва по позиции.
